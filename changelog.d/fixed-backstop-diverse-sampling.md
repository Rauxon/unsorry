The independent-check backstop now samples recently-merged proofs **randomly** (`shuf -n N`) rather than the alphabetically-first N (`sort | head`), so a daily run covers a diverse slice of proof shapes and successive runs rotate over the whole window — instead of repeatedly checking near-identical `AltGeometricRatio*` proofs (ADR-096 / SPEC-096-A §5b). `shuf -n` also emits N and exits cleanly, removing the head-induced SIGPIPE the previous version had to work around.
