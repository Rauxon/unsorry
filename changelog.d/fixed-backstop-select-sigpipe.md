Fixed the independent-check backstop's proof-sampling step aborting with `sort: write failed: Broken pipe` (ADR-096 / SPEC-096-A §5b). `head -n N` closes the pipe after N lines, so when more proofs matched, upstream `sort` got SIGPIPE and `pipefail` failed the step. The sampling pipeline now tolerates it (`{ … ; } || true`), so a full window legitimately yields the first N and an empty window falls through to the spread sample.
