The advisory independent check (ADR-096) now surfaces its verdict **on the proof PR**, not just the local log. After a proof passes the kernel-diverse `nanoda` re-check, the verdict (status, declarations checked, target-confirmed, timing, export size) rides the proof commit as an `Independent-Check:` git trailer; the dispatcher reads it back from the branch and appends an _"Independent check (advisory, ADR-096): …"_ line to the PR body (the direct/fork path injects it in-process). So an independently-checked proof is now distinguishable on GitHub. It carries no Gate B / soundness weight — the trailer leaves the prove-commit subject untouched and a normal proof shows nothing extra; ADR-049's p=1 Lean gate is unchanged.
