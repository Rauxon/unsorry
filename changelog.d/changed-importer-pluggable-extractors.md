`import_benchmark` now supports **pluggable per-suite extractors** via `--extractor`: `theorem-named` (PutnamBench / miniF2F / CombiBench — the theorem name is the goal id) and `imolean` (IMOLean — the goal id comes from the *filename*, and the per-problem `namespace` wrapper is dropped). Adds `--exclude` to skip a suite's errata list. This generalises the importer beyond PutnamBench so M8b–M8d (miniF2F, CombiBench, historical/olympiad) can be ingested with only an extractor + a license check. (#5643.)
