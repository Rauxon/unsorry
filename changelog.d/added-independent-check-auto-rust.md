`./swarm/run.sh --independent-check` is now fully self-contained (ADR-096): if Rust/cargo (needed to build `nanoda`) is missing, the setup helper installs it automatically via rustup (non-interactive, minimal profile) before building — so the single `run.sh` command is all a contributor needs, even on a machine without Rust. Idempotent (a prior rustup install is just re-PATH'd); the PATH change is local to the build (the compiled `nanoda_bin` runs standalone). If `curl` is also absent it degrades to a clear message and proves without the check (still non-gating).
